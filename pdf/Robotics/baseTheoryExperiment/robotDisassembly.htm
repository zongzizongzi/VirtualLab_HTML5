工业机器人拆装实验指导书

1)    实验目的

本实验目的是了解机器人结构，了解每个连杆的运动特征，深入理解连杆坐标系与机器人坐标变换的关系。

1)    实验原理

工业机器人拆装实验要求学生将机器人模型以机械臂为单位进行拆解和组装，进一步观察机器人的机械结构。

2)    实验过程

图1 机器人拆装示意图

       本实验操作过程为：

（1）   打开实验界面；

（2）   观察机器人整体结构，识别各个机械臂单元；

（3）   鼠标拖动虚拟机器人模型的机械臂构件，将其逐一拆解；

（4）   观察独立的机械臂结构；

（5）   分析机械臂的运动特性和连杆坐标系；

（6）   鼠标拖动虚拟机器人模型分离的机械臂构件，将其装配回原位。

(7)    装配完成之后，进入下一环节

3)    扩展训练

（1）   根据实验2.1.1得到的D-H参数，写出每个运动单元的连杆变换矩阵。

（2）   思考连杆变换矩阵与机械手变换矩阵、机械臂单元运动与机器人整体运动之间的联系。

附录1：工业机器人常用控制指令

MoveL - 使机械臂沿直线移动
基本用途：

MoveL用于将工具中心点沿直线移动至给定目的。当TCP保持固定时，则该指令亦可用于调整工具方位。

该指令的基本范例及说明如下：

示例：MoveL p1, v1000, z30, tool2;

说明：工具Tool2的工具中心点将直线运动至位置p1，其速度数据为v1000。
MoveC―让机器人做圆周运动

基本用途：

该指令用来让机器人 TCP 沿圆周运动到一个给定的目标点。在运动过程中，相对圆的方向通常保持不变。

该指令的基本范例及说明如下：

示例： Move p1, p2, v500, tool2;

说明：工具tool2的工具中心点 圆周运动到 p2，速度数据为v500.圆由开始点、中间点 p1 和目标点 p2 确定

MoveJ - 通过关节移动，移动机械臂

基本用途：

当该运动无须位于直线中时，MoveJ用于将机械臂迅速地从一点移动至另一点。机械臂和外轴沿非线性路径运动至目的位置。所有轴均同时达到目的位置。

该指令的基本范例及说明如下：

示例：MoveJ p1, v500, z30, tool2;

说明：将工具tool2的工具中心点沿非线性路径移动至位置p1，其速度数据为v500。

Reset - 重置数字信号输出信号

基本用途：

Reset为，用于将数字信号输出信号的值重置为零。

该指令的基本范例及说明如下：

示例：Reset do15;

说明：将信号do15设置为0

Set - 设置数字信号输出信号
基本用途：

Set用于将数字信号输出信号的值设置为一。

该指令的基本范例及说明如下：

示例：Set do15;

说明：将信号do15设置为1
