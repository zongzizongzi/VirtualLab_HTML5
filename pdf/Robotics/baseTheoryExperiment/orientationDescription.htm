刚体方位的描述实验指导书

1)    实验目的

本实验以三维空间模型辅助数学模型的展示，用三维模型展示不同刚体位姿描述方法中每个参数的意义，加深学生对于方位描述的理解。

2)    实验原理

除了用旋转矩阵描述物体的方向，常用的方位描述方法还有：RPY角、欧拉角、轴_角、和四元数。这几种方法都可以跟旋转矩阵相互转换。本实验包含几个小实验，将分别向学生介绍这几种方法的物理意义及其与旋转矩阵的转换关系。

（1）   RPY角表示位姿

RPY角是相对于固定（参考）坐标系旋转的。最初固定坐标系{A}和运动坐标系重合，先将绕的X轴旋转，称为偏转(Yaw)；然后绕{A}的Y轴旋转，称为俯仰(Pitch)；最后绕的Z轴旋转，称为回转(Roll)，就能旋转到当前姿态,如图1 RPY角所示。

（2）   欧拉角表示位姿

欧拉角是相对于运动坐标系旋转的。本实验中采取欧拉角，其法则如下：最初固定坐标系和运动坐标系重合，先将绕自身的Z轴旋转，然后绕自身Y轴旋转，最后绕自身Z轴旋转，就能旋转到当前姿态。

这种描述方法中的各次转动都是相对于运动坐标系的某轴进行的，而不是相对于固定的参考系。这样的三次转动称为欧拉角，又因转动的顺序是绕轴、轴和轴，故称这种描述方法为欧拉角。

（4）   四元数表示位姿

欧拉角容易出现的问题有：不易在任意方向的旋转轴插值；方向节死锁；旋转的次序无法确定。

3)    实验过程

图3 刚体方位描述实验界面

本实验操作过程如下：

（1）   在实验界面上选择描述方式为“RPY角”，变换方程显示由RPY角转换为旋转矩阵的公式；

（2）   拖动控制栏的滑动条（或在对应输入框内填写数字）依次控制三维显示区的运动坐标系（高亮显示）模型绕参考系的X、Y、Z轴转动；

（3）   到达目标位姿后，根据转换公式，在旋转矩阵填中写对应的计算结果。

（4）   在实验界面上选择描述方式为“欧拉角”，变换方程显示由欧拉角转换为旋转矩阵的公式；

（5）   拖动控制栏的滑动条（或在对应输入框内填写数字）依次控制三维显示区的运动坐标系（高亮显示）模型绕自身的Z、Y、Z轴转动；

（6）   到达目标位姿后，根据转换公式，在旋转矩阵中填写对应的计算结果。

（7）   在实验界面上选择描述方式为“轴-角”，变换方程显示由轴-角模式转换为旋转矩阵的公式；

（8）   拖动控制栏的滑动条（或在对应输入框内填写数字），改变旋转轴k的x、y、z分量；

（9）   点击归一化旋转轴，将三个分量组成的转轴向量归一化为单位向量，并在三维显示区显示三维旋转轴的位置；

（10）   拖动控制栏的滑动条（或在对应输入框内填写数字），改变旋转角度θ，控制三维显示区的运动坐标系模型绕k轴旋转角度θ；

（11）   到达目标位姿后，根据转换公式，在旋转矩阵中填写对应的计算结果。

（12）   实验界面上选择描述方式为“四元数”，变换方程显示由四元数转换为旋转矩阵的公式。

（13）   拖动控制栏的滑动条（或在对应输入框内填写数字），改变四元数的四个分量；

（14）   点击“归一化四元数”，将四个分量组成的四维空间向量归一化为单位向量，同时三维显示区的运动坐标系模型旋转至指定位姿。

（15）   根据转换公式，在旋转矩阵中填写对应的计算结果。

（16） 提交实验结果，结果正确方能进入下一环节。

4)    扩展训练

各种位姿描述方法通过与旋转矩阵相联系，试推导各种描述方法之间的转换关系。

 附录1：工业机器人常用控制指令

MoveL - 使机械臂沿直线移动
基本用途：

MoveL用于将工具中心点沿直线移动至给定目的。当TCP保持固定时，则该指令亦可用于调整工具方位。

该指令的基本范例及说明如下：

示例：MoveL p1, v1000, z30, tool2;

说明：工具Tool2的工具中心点将直线运动至位置p1，其速度数据为v1000。
MoveC―让机器人做圆周运动

基本用途：

该指令用来让机器人 TCP 沿圆周运动到一个给定的目标点。在运动过程中，相对圆的方向通常保持不变。

该指令的基本范例及说明如下：

示例： Move p1, p2, v500, tool2;

说明：工具tool2的工具中心点 圆周运动到 p2，速度数据为v500.圆由开始点、中间点 p1 和目标点 p2 确定

MoveJ - 通过关节移动，移动机械臂

基本用途：

当该运动无须位于直线中时，MoveJ用于将机械臂迅速地从一点移动至另一点。机械臂和外轴沿非线性路径运动至目的位置。所有轴均同时达到目的位置。

该指令的基本范例及说明如下：

示例：MoveJ p1, v500, z30, tool2;

说明：将工具tool2的工具中心点沿非线性路径移动至位置p1，其速度数据为v500。

Reset - 重置数字信号输出信号

基本用途：

Reset为，用于将数字信号输出信号的值重置为零。

该指令的基本范例及说明如下：

示例：Reset do15;

说明：将信号do15设置为0

Set - 设置数字信号输出信号
基本用途：

Set用于将数字信号输出信号的值设置为一。

该指令的基本范例及说明如下：

示例：Set do15;

说明：将信号do15设置为1
