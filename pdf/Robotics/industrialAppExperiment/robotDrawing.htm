工业机器人绘图实验指导书

1)    实验目的

工业机器人的绘图实验要求学生设计一个平面图案，并在虚拟实验平台上控制机器人绘制出此图案，目的是让学生进一步熟悉对机器人控制过程和控制指令。

1)    实验原理

工业生产中经常利用工业机器人进行喷涂、焊接或雕刻工作，我们将这些工作统一抽象化为绘图实验。与码垛实验不同，绘图实验在绘制过程中不仅要求运动的关键点准确，还对两关键点之间的路径有要求。

本实验的具体要求如下：

（1）   以XY平面（即三维场景中的田字平面）作为绘图的纸面范围，画笔接触纸面过程中的路径线条作为有效绘图线条；

（2）   绘制的图案具有一定意义；

（3）   绘图过程中画笔保持竖直；

（4）   在画笔脱离纸面（即实验中的）之后的动作不作要求。

2)    实验过程

图1 机器人绘图实验界面

本实验操作过程为：

（1）   设计平面图案，并用直线和圆弧拟合该图案，获取关键点的坐标（此过程可以利用其他软件或程序进行）；

（2）   打开实验界面；

（3）   在加载工件的下拉框中选择“huabi”,为机器人加载画笔工具；

（4）   示教机器人，在每个运动节点添加示教点，或将软件生成的示教点坐标按照示教点格式写入示教输入框中；

（5）   编写运动控制指令；

（6）   运行控制指令并调试，直至机器人能正确地绘制出所设计的图案。

3)    扩展训练

（1）   软件生成的图案示教点数量庞大，使得示教和编写程序的过程过于繁琐，可以采取哪些措施简化？

尝试在XZ、YZ或倾斜平面上绘制图形。

附录1：工业机器人常用控制指令

MoveL - 使机械臂沿直线移动
基本用途：

MoveL用于将工具中心点沿直线移动至给定目的。当TCP保持固定时，则该指令亦可用于调整工具方位。

该指令的基本范例及说明如下：

示例：MoveL p1, v1000, z30, tool2;

说明：工具Tool2的工具中心点将直线运动至位置p1，其速度数据为v1000。
MoveC―让机器人做圆周运动

基本用途：

该指令用来让机器人 TCP 沿圆周运动到一个给定的目标点。在运动过程中，相对圆的方向通常保持不变。

该指令的基本范例及说明如下：

示例： Move p1, p2, v500, tool2;

说明：工具tool2的工具中心点 圆周运动到 p2，速度数据为v500.圆由开始点、中间点 p1 和目标点 p2 确定

MoveJ - 通过关节移动，移动机械臂

基本用途：

当该运动无须位于直线中时，MoveJ用于将机械臂迅速地从一点移动至另一点。机械臂和外轴沿非线性路径运动至目的位置。所有轴均同时达到目的位置。

该指令的基本范例及说明如下：

示例：MoveJ p1, v500, z30, tool2;

说明：将工具tool2的工具中心点沿非线性路径移动至位置p1，其速度数据为v500。

Reset - 重置数字信号输出信号

基本用途：

Reset为，用于将数字信号输出信号的值重置为零。

该指令的基本范例及说明如下：

示例：Reset do15;

说明：将信号do15设置为0

Set - 设置数字信号输出信号
基本用途：

Set用于将数字信号输出信号的值设置为一。

该指令的基本范例及说明如下：

示例：Set do15;

说明：将信号do15设置为1
